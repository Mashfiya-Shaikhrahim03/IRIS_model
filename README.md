# 🌸 Iris Flower Classification

A machine learning project that classifies iris flowers into different species based on their physical measurements.

</div>

## 📊 Dataset Information

The dataset contains flower measurements such as:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

**Target Variable:** Iris Species

* Setosa
* Versicolor
* Virginica

## 🔍 Project Workflow

* Explored and analyzed the dataset
* Checked feature distributions
* Visualized relationships between features
* Prepared data for training
* Split data into training and testing sets
* Trained multiple classification models
* Compared model performance

## 🤖 Models Used

* K-Nearest Neighbors (KNN)
* Gaussian Naive Bayes (GNB)
* Logistic Regression

## 📈 Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Classification Report

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 🎯 Project Outcome

Successfully built and compared multiple classification models for predicting iris flower species. This project provided hands-on experience with supervised machine learning and model evaluation techniques.
